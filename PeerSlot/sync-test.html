‚ú® How It Works

  • When you add, edit, or delete an availability slot, the availability-ui.js module dispatches a custom event called availabilitySlotsChanged
  • The dashboard-calendar.js module listens for this event using window.addEventListener()
  • When the event is received, the calendar automatically calls loadAvailabilityToCalendar() to refresh its display
  • This creates a real-time, decoupled synchronization between components without tight coupling
  • The pattern uses the Observer/Publisher-Subscriber design pattern via browser's native CustomEvent API
