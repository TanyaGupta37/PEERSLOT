🔄 Data Collection Process

How Rewards Are Calculated:
  1. On Rewards Page Load: System automatically fetches all completed sessions for the current user
  2. Query matchRequests Collection:
    • Sessions where user is requesterId = User was RECEIVING help (20 tokens each)
    • Sessions where user is slotOwnerId = User was HELPING (50 tokens each)
  3. Calculate Total Tokens: Sum tokens from both roles
  4. Determine Badges: Compare against milestone thresholds
  5. Update User Document: Store calculated rewards in user profile for future reference
  6. Display UI: Show total tokens, earned badges, and progress to next milestone
